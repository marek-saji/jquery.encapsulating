encapsulating textarea

a jQuery plugin

You can usually see these in tag input fields. Also known as token fields and typeahead (in connection with autocomplete).

  • download
  • clone from github

try it out

Use comma (,) or new lines (enter) to separate items. You can also navigate with tab and arrow keys.

usage

Add this in your <head />: 

<script type="text/javascript" src="http://code.jquery.com/jquery-1.4.4.min.js"><>
<script type="text/javascript" src="jquery.encapsulating.js"><>
<link rel="stylesheet" href="jquery.encapsulating.css" />
<script type="text/javascript">
    $('textarea').encapsulating();
</script>

configuration

It's configurable by:

  • invoke arguments 
    $('textarea').encapsulating({removeText: '-'});
  • element's jQuery's element data, including (since jQuery 1.4.3) data-encapsulating argument 
    <textarea data-encapsulating='{"debug":true}'></textarea>

Values specified as element data override invoke configuration.
